Dofus reader app
Приложение для чтения на иностранных языках, где перевод и словарь встроены прямо в текст. Основная идея приложения — постепенное привыкание к языку через чтение. По мере того как пользователь сохраняет и запоминает слова, необходимость в переводе снижается, и оригинальный текст становится всё более понятным.
